A12及以上

1，下载misaka，爱思助手签名安装 
如果签名失败，在网页登陆一下appid
3，开启misaka应用
设置->通用->VPN与设备管理 点 信任
设置->隐私与安全性->开发者模式 打开重启
4，安装巨魔TrollStore
5，打开TrollStore 完成安装
dltd安装，这里因为是访问github，很大概率会下载失败，一直重试


A10

1，用爱思助手选择 https://github.com/palera1n/palen1x/releases c-palen1x-v1.0.10-2-amd64.iso 制作U盘
2，在intel机型上从U盘启动按步骤安装palen1
3，打开palen1添加源https://havoc.app/安装 TrollStore Helper
4，打开 TrollStore Helper 安装 TrollStore
5，打开 TrollStore 完成安装
